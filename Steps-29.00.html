Creating Attrition by Year Line and stacked column chart 29
  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Pane:
  • X-axis = Year
  • Column y-axis= Count of Employment Status column
  • Line  y-axis = .AverageYearofService measure rename (Yr of Service), .AverageHourlyRate measure rename (Hourly Rate)
  • Column Legend = Employment Status column
  • Tooltips = .AverageYearsofService, .AverageHourlyRate
Format Visual Pane:

General:

  • Size and Position: Height=318, Width=548, Horizontal=192, Vertical=76
  •  Title=On, Text= Attrition By Year, Font= Segoe Bold 14, Text color= white, Background color=#506D97, Horizontal alignment=Center
  • Effects: Visual Border On = Color = #CCCCCC 1px, 1px, Background=Off
Visual:

X-axis: Categorical, Values = On, Font=Segoe Bold 9; Color=Black; Title=Off, Max Height= 25%

  • Y-axis: Values=On, Font=Segoe UI Bold 9; Color=Black; Title=Off
  • Secondary Y-axis: Value=On, Font=Segoe Bold 9, Color=Black
  • Legend: On, Position=Top Left, Style=Marker, Match line color=on, Text=Seqoe 10,Title=On, Text= Emp. Status
  • GridLines: Horizontal=On, Color=#C8C6C4, Dotted, 1px
  • Columns: Color = Series A = Color= #3257A8, Series I = Color= #A6BDDF
  • Lines: Series = Yr of Service, Show Series = On, Line: Line Style= Solid, Round,3px, Color= #DD6B7F, Series = Hourly Rate, Color = #8B3D88
  • Shade Area: On, Series = Yr of Service and Hourly Rate, Show Series = On, Match line color = On
  • Total Labels: Font=Segoe Bold 9, Color= Black, Background On: Color #7981E8, 30%
