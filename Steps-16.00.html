Create MeasureSubKpi Table for your YOY Change Chart 16
  1. On Power BI Desktop ribbon, Click on Enter data
  2. Name the Table “MeasureSubKpi”
  3. Create a column to add the Measures and the index value, then load the table.
  4. Create a new measure in the MeasureSubKpi Table
  5. On the Dax formula bar type 
MeasureValue = SWITCH(SELECTEDVALUE(MeasureSubKpi[Index]),1, [.YoYTotalEmployee],2, [.YoYActiveEmployee],3, [.YoYAttritionRate],4, [.YoYAvgYearsofService],5, [.YoYAvgHourlyRate], 6, [.YoYAvgSalaryExpense],BLANK())
