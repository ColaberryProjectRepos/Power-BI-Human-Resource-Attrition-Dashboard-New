Colaberry Logo

Download the "Colaberry-Logo" file by clicking the red icon above
  1. From Power BI Desktop ribbon under Insert, click on Image 
  2. Select the colaberry logo from the downloadable file provided in this project
  3. Under the Format Image pane, click on Image and fill in the figures provided below
  •      Size: Height=79, Width = 237
  •      Position: Horizontal=1041, Vertical=0
  •      Background = on, color = White


Title

  1. From Power BI Desktop ribbon under Insert, click on Text box from Elements
  2. Type in 'Human Resource Attrition' in the text box
  3. Select center alignment for the text (Font = Segoe UI, Font size = 32, Font Color = #3257A8) 
  4. On Format text box Pane, on visual option, fill in the figures provided below
  •  Size: Height=79, Width=805
  •  Position: Horizontal=236, Vertical=0                                                             
  • Background = on, color = White
