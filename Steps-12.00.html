1. Download the City-Parish_Employees_2024 file by clicking the red icon above

2. Open Power BI Desktop

3.  Click on Get data ribbon.

4. Select Text/CSV and then click connect

5. Open saved file, choose “City-Parish_Employees_2024”CSV file and click on Open

6. A preview of the data will be displayed.

7. Ensure data is correct and clean as necessary using the Power Query Editor tools, also create the following Calculated Columns.

  • Attrition Count = Table.AddColumn(#"Changed Type", "Attrition Count", each if [EMPLOYMENT STATUS] = "I" then 1 else 0)

  • Birthdate = Table.AddColumn(#"Added Conditional Column", "BIRTHDATE", each Date.AddYears([CURRENT HIRE DATE], -19))

  • Age = Table.AddColumn(#"Changed Type1", "Age", each Date.From(DateTime.LocalNow()) - [BIRTHDATE], type duration)

  • Age Group = Table.AddColumn(#"Reordered Columns", "AGE GROUP", each if [Age] >= 60 then "60 And Above" else if [Age] >= 50 then "50-60" else if [Age] >= 40 then "40-50" else if [Age] >= 30 then "30-40" else "30 or Below")

  • Employee Cnt = Table.AddColumn(#"Removed Errors", "Employee Cnt", each if [EMPLOYMENT STATUS] = "I" then 1 else if [EMPLOYMENT STATUS] = "A" then 1 else if [EMPLOYMENT STATUS] = "B" then 1 else null)

  • Duplicate UniqueID= Table.DuplicateColumn(#"Changed Type3", "UNIQUEID", "UNIQUEID - Copy")

  • ID = Table.AddIndexColumn(#"Duplicated Column", "Index", 1, 1, Int64.Type)
8. Once you have completed any transformation click close and apply
