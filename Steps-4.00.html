Methodology 4

Data Preparation:

Clean and aggregate human resource data for analysis, ensuring its accuracy by eliminating duplicates and addressing missing values.



Model Selection:

GradientBoostingClassifier was selected as the optimal model for this human resource attrition project due to it often providing strong predictive performance, particularly on structured/tabular data, due to its ability to model complex relationships between features. For this project, the model performs consistently across all these metrics, with very close values (around 84%), indicating a balanced performance in correctly predicting both positive and negative instances.



Model Development:

The GradientBoostingClassifier model was trained and optimized using human resource attrition data. The dataset was split into training and testing sets. The training set enabled the model to identify patterns, while the testing set assessed its ability to generalize to unseen data. This approach helps prevent overfitting, ensuring the model doesn’t perform well only on the training data but also on new, unseen data. By using a test set, the model’s predictions remain unbiased and reliable.
