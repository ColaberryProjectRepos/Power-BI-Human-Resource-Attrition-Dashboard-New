Creating Total Employees KPI

  1. Click on an empty space on the canvas, and from the Build pane select Card visual
  2. On the build pane drag and drop the Total Employees measure 

Visual Format

Callout value (number) - Font = Segou Bold 20, color =#7981E8

Category Label =off

 
General

Size - Height =98, width=192

Position - Horizontal =0, vertical=77

Title on - Total Employees, Font = Segou Bold 12, Text color #FFFFFF, Background=506D97, Horizontal alignment= center

Effects – Background = off, Visual Border = on, Color #CCCCCC, 1px

Data Format – Format Options = whole number, Thousand Separator = on, Decimal Places =0
