YOYTotalEmployee arrow indicator
  1. Click on an empty space on the canvas, and from the visualizations pane select Visual Card
  2. Drag the .YOYTotalEmpArrow measure to the fields value box or drag to the visual card
General

Size: Height=46, Width=51

Horizontal=138, Vertical=111

Effects – Background = off

Visual:

Callout Value: Font=Segou Bold 20, Font color = FX (click on the function tab and fill up values as displayed below)

Format Style = Rules, Field = .YOYTotalEmployee, IF >=0 and <100 = #00F71D, IF >= -100 and <0 = #F91702

Category Label =  Off
