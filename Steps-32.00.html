Build Visual Pane:
  • Category = MeasureName
  • y-axis= MeasureValue
  • No Breakdown
Format Visual Pane:

General:

  • Size and Position: Height=318, Width=538, Horizontal=740, Vertical=76
  • Title=On, Text= YOY Change, Font= Segoe Bold 14, Text color= white, Background color=#506D97, Horizontal alignment=Center
  • Effects: Visual Border On = Color = #CCCCCC 1px, 1px, Background=Off
  • Data Format: MeasureValue=Percentage, Decimal Places=0
Visual:

  • X-axis: Values = On, Font=Segoe Bold 9; Color=Black; Title=Off, Max Height= 25%
  • Y-axis: Values=On, Font=Segoe UI Bold 9; Color=Black; Title=Off
  • Legend: On, Position=Top Left, Text=Seqoe Bold 10, Title=Off
  • GridLines: Horizontal=On, Color=#C8C6C4, Dotted, 1px
  •  Columns: Increase =Color= #1AAB40, Decrease = Color= #D64554, show total column=off
  • Data Labels: On, Horizontal, Auto
